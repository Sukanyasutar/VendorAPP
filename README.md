# VendorApp

A comprehensive **Vendor & Customer Collaboration Portal** (PO & Invoice Management System) designed to streamline the procurement lifecycle, invoice processing, and communication between vendors, customers, and administrators.

## 🚀 Key Features

### 🏢 Multi-Role Access Control
- **Vendor Portal:** Manage purchase orders, submit and track invoices, upload Proof of Delivery (POD), and resolve invoice/PO rejections.
- **Customer Portal:** Review pending invoices, approve/reject submissions, and check vendor activity dashboards.
- **Admin Dashboard:** Monitor system-wide transactions, manage users, and generate overview reports.

### 📋 Core Modules
* **Purchase Order (PO) Management:** View active purchase orders and create/issue new POs seamlessly.
* **Invoice & Billing Lifecycle:** Direct invoice creation, submission, status tracking, and automated matching.
* **Proof of Delivery (POD) Center:** Secure file upload and tracking for POD documents to verify goods receipt.
* **Rejection Resolution:** Direct communication channel to discuss and resolve discrepancies on rejected invoices.
* **Reports & Analytics:** Built-in charts and tables tracking transaction statuses, payment history, and operational performance.

## 💻 Tech Stack
- **Frontend Framework:** React 18 with TypeScript
- **Routing:** React Router v7
- **Styling & Components:** TailwindCSS, Tailwind-merge, shadcn/ui-inspired styling, and Material UI (MUI) components/icons
- **Animations:** Motion (Framer Motion)
- **Charts:** Recharts
- **Icons:** Lucide React

## ⚙️ Running the Code

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Access the application:**
   Open your browser and navigate to `http://localhost:5173`.
